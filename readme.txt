Build powerful layouts with blocks, patterns & templates using GL Layout Builder.

== Description ==

GL Layout Builder enhances your Gutenberg editing experience by providing a collection of custom blocks, powerful extensions, and enhancements for core blocks. It also includes a library of pre-designed patterns and templates to speed up your workflow.

With GL Layout Builder, you can easily create stunning layouts and professional-looking websites without needing to code.

**Key Features:**

*   **Custom Blocks:** A set of useful blocks to extend your content creation capabilities.
*   **Core Block Enhancements:** Adds extra features and controls to existing WordPress core blocks.
*   **Pattern Library:** Access a wide range of professionally designed patterns.
*   **Template Library:** Ready-to-use templates for various page types.
*   **Performance Optimized:** Lightweight and fast, ensuring your site remains performant.

== Installation ==

1.  Upload the plugin files to the `/wp-content/plugins/gl-layout-builder` directory, or install the plugin through the WordPress plugins screen directly.
2.  Activate the plugin through the 'Plugins' screen in WordPress.
3.  Start using the new blocks and patterns in the Gutenberg editor.

== Frequently Asked Questions ==

= Does this plugin require a specific theme? =

No, GL Layout Builder is designed to work with any WordPress theme that supports the Gutenberg editor. However, it works best with Full Site Editing (FSE) themes.

= Can I customize the patterns? =

Yes, all patterns provided by GL Layout Builder are fully customizable using the standard Gutenberg block settings.
